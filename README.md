# 🌿 KeenKeeper — Keep Your Friendships Alive

## 📖 Description

KeenKeeper is a modern React-based web application designed to help users maintain and nurture meaningful friendships. It allows users to track interactions, set communication goals, and visualize relationship activity through a clean and responsive interface.

---

## 🚀 Technologies Used

* ⚛️ React.js
* 🧭 React Router DOM
* 🎨 Tailwind CSS
* 📊 Recharts
* 🔔 React Toastify
* 🧩 Lucide React Icons

---

## ✨ Key Features

### 1. 👥 Friend Management

* View all friends in a responsive card layout
* Each friend includes profile info, contact status, tags, and last interaction

### 2. ⚡ Interaction Tracking (Timeline)

* Log Call, Text, and Video interactions instantly
* Timeline updates dynamically with date and activity type
* Toast notifications for user actions

### 3. 📊 Analytics Dashboard

* Visualize interaction types using a Pie Chart
* Get insights into communication patterns

---

## 📱 Responsive Design

* Fully responsive across mobile, tablet, and desktop
* Clean UI based on Figma design

---

## 🧠 Additional Features

* Dynamic routing with React Router
* Context API for global timeline state
* Loading spinner while fetching data
* Custom 404 Not Found page
